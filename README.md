# Run instruction </br>

### 1. Install docker on your PC 

[install docker](https://www.docker.com/get-started/)

### 2. Open terminal and enter below command 

```docker run --name architechDb -e POSTGRES_PASSWORD=postgres -p 5432:5432 -d postgres```
